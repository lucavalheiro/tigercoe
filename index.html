Storage

Soluções de Storage

  • IBM FlashSystem
    Armazenamento em flash de alta performance, com baixa latência e compressão inline.
    Proteção com IBM Safeguarded Copy e IBM FlashCore Modules.
    Suporte a multicloud, containers (Red Hat OpenShift) e virtualização.
  • IBM Storage Fusion
    Plataforma de armazenamento para containers e ambientes Kubernetes.
    Integrada ao Red Hat OpenShift com provisionamento automático, snapshots e mobilidade de dados.
  • IBM Storage Ceph
    Solução Software Defined Storage (SDS) para blocos, arquivos e objetos.
    Baseada em código aberto (Ceph), ideal para ambientes escaláveis, cloud-native e workloads de IA.
  • IBM Storage Scale (antigo Spectrum Scale)
    Sistema de arquivos paralelos para dados não estruturados em escala massiva.
    Suporte a HPC, IA, análise avançada e bancos de dados distribuídos.
  • IBM Cloud Object Storage
    Armazenamento de objetos escalável, durável e seguro, disponível on-premises e em cloud.
    Ideal para backup, arquivamento, big data, repositórios de IA e dados não estruturados.
  • IBM Storage Defender (novo lançamento em 2024/2025)
    Plataforma integrada para resiliência cibernética, backup, recuperação e proteção de dados contra ransomware.
  • IBM Spectrum Protect
    Solução tradicional de backup e recuperação, com suporte a ambientes físicos, virtuais e em nuvem.
    Suporte avançado a deduplicação, automação e escalabilidade.
  • IBM Storage Insights
    Ferramenta SaaS para monitoramento, análise e otimização do ambiente de storage IBM.
    Proativa, com alertas e recomendações baseadas em IA.
  • IBM Tape Storage (TS Series)
    Soluções de fita magnética para arquivamento de longo prazo com baixíssimo custo e alta durabilidade.
    Compatível com LTO Ultrium e tecnologias WORM.
Power Systems

IBM Power Systems Bare Metal na IBM Cloud

  • IBM Power Systems Virtual Server (dedicado físico por partição)
    Infraestrutura Power9 ou Power10, com partições lógicas (LPARs) sob seu controle.
    Baseado em hardware físico real (não emulação x86).
    Opção de uso dedicado (bare metal compartilhado entre suas próprias LPARs).
  • Sistemas operacionais suportados
    • IBM AIX
    • IBM i (AS/400)
    • Linux on Power (RHEL, SUSE)
  • Recursos técnicos
    • CPU: POWER9 ou POWER10
    • Suporte a VIOS (Virtual I/O Server), NIM, HMC virtual
    • Conectividade com serviços IBM Cloud via VPC ou Classic Infrastructure
    • Suporte a backup, snapshot e replicação
    • Integração com serviços como IBM Cloud Object Storage, Block Storage, e Spectrum Protect
  • Casos de uso típicos
    • Modernização de workloads legados em AIX e IBM i
    • Bancos de dados Db2, Oracle, SAP on AIX
    • Aplicações ERP críticas
    • Ambientes híbridos com IBM Power e x86
  • Diferenciais da oferta Power na IBM Cloud
    • Total controle sobre a infraestrutura (como on-premise)
    • Evita refatoração de código legado (AIX, IBM i)
    • Pode ser integrado ao restante da IBM Cloud (Kubernetes, Watsonx, etc.)
    • Preço por hora ou mensal com escalabilidade
